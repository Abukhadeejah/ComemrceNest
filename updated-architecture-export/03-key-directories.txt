KEY DIRECTORIES & MODULES
======================================

/src/app - NEXT.JS APP ROUTER
--------------------------------------
The main application structure using Next.js App Router:

(admin)                    - Admin dashboard wrapper
├── admin/                 - Main admin pages
│   ├── products/          - Product CRUD & management
│   ├── categories/        - Category management
│   ├── orders/            - Order processing
│   ├── customers/         - Customer database
│   ├── inventory/         - Stock management
│   └── settings/          - System configuration

(site)                     - Customer storefront wrapper
├── bluebell/              - First tenant store
├── senlysh/               - Second tenant store
└── [slug]/                - Dynamic product/portfolio pages

(tenant-admin)             - Multi-tenant admin
└── [tenant]/              - Dynamic tenant routes

api/                       - Backend API routes
├── admin/                 - Admin endpoints
├── auth/                  - Authentication (Supabase)
├── checkout/              - Cart & checkout
├── webhooks/              - Payment webhooks (Razorpay, PhonePe)
└── site/                  - Frontend APIs
