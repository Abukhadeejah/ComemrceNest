KEY DIRECTORIES & MODULES
======================================

/src/app - NEXT.JS APP ROUTER
--------------------------------------
The main application structure using Next.js App Router:

(admin)                    - Admin dashboard wrapper
├── admin/                 - Main admin pages
│   ├── products/          - Product CRUD & management
│   ├── categories/        - Category management
│   ├── orders/            - Order processing
│   ├── customers/         - Customer database
│   ├── inventory/         - Stock management
│   └── settings/          - System configuration

(site)                     - Customer storefront wrapper
├── bluebell/              - First tenant store
├── senlysh/               - Second tenant store
└── [slug]/                - Dynamic product/portfolio pages

(tenant-admin)             - Multi-tenant admin
└── [tenant]/              - Dynamic tenant routes

api/                       - Backend API routes
├── admin/                 - Admin endpoints
│   ├── attributes/        - Product attributes CRUD
│   ├── categories/        - Category management
│   ├── coupons/           - Coupon system
│   ├── orders/            - Order management
│   ├── products/          - Product CRUD & drafts
│   ├── settings/          - System settings
│   └── upload/            - File uploads
├── auth/                  - Authentication (Supabase)
│   ├── [...nextauth]/     - NextAuth handler
│   ├── admin-tenant/      - Tenant admin auth
│   ├── callback/          - Auth callbacks
│   ├── check-tenant-access/ - Tenant access validation
│   ├── signout/           - Sign out
│   └── whoami/            - User info
├── checkout/              - Cart & checkout
├── coupons/               - Coupon validation
├── customers/             - Customer management
│   ├── addresses/         - Address management
│   ├── profile/           - Profile updates
│   ├── register/          - Registration
│   └── wallet/            - Wallet operations
├── debug/                 - Debug endpoints
├── health/                - Health check
├── orders/                - Order processing
├── product-drafts/        - Draft management
├── products/              - Product APIs
├── rewards/               - Cashback system
│   ├── calculate/         - Cashback calculation
│   └── history/           - Reward history
├── site/                  - Frontend APIs
│   ├── categories/        - Public categories
│   └── tags/              - Product tags
├── wallet/                - Wallet system
│   ├── cashback-stats/    - Cashback statistics
│   ├── preview-cashback/  - Preview calculations
│   └── transactions/      - Transaction history
└── webhooks/              - Payment webhooks
    ├── juspay/            - Juspay webhooks (legacy)
    └── phonepe/           - PhonePe webhooks
