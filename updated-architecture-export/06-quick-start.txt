QUICK START & REFERENCE GUIDE
======================================

INSTALLATION & SETUP
--------------------------------------
1. Install dependencies:
   npm install

2. Configure environment (.env.local):
   SUPABASE_URL=your_url
   SUPABASE_ANON_KEY=your_key
   SUPABASE_SERVICE_ROLE_KEY=your_key
   RAZORPAY_KEY_ID=your_id
   RAZORPAY_KEY_SECRET=your_secret
   PHONEPE_MERCHANT_ID=your_id
   PHONEPE_API_KEY=your_key

3. Start development server:
   npm run dev

4. Open browser:
   http://localhost:3000

DEBUGGING TIPS
--------------------------------------

Check Logs:
  1. Browser console (F12)
  2. Server terminal output
  3. Supabase dashboard logs

Common Issues:
  ✓ CORS errors → Check API origins
  ✓ Auth errors → Check .env variables
  ✓ Database errors → Check RLS policies
  ✓ Payment errors → Check webhook configurations

Database Debugging:
  - Use Supabase dashboard to view data
  - Check RLS policies for permission issues
  - Run migrations if schema changes
