CONFIGURATION FILES REFERENCE
======================================

MAIN CONFIGS
--------------------------------------

package.json
  - Node.js dependencies (Next.js, React, TailwindCSS, etc.)
  - Build & dev scripts
  - Project metadata

tsconfig.json
  - TypeScript compilation settings
  - Path aliases (@/ for src/)
  - Strict type checking

next.config.ts
  - Next.js build optimization
  - Image optimization
  - Environment variables

tailwind.config.ts
  - TailwindCSS theme customization
  - Color schemes
  - Responsive breakpoints

postcss.config.js
  - PostCSS plugin configuration
  - TailwindCSS integration

eslint.config.mjs
  - Code style & quality rules
  - TypeScript support
  - React rules

API & DEPLOYMENT
--------------------------------------

vercel.json
  - Vercel deployment configuration
  - Build settings
  - Environment setup

playwright.config.ts
  - E2E test configuration
  - Browser settings
  - Test timeouts
